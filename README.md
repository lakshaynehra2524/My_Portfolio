# Lakshay — Portfolio (FastAPI)

A FastAPI backend serving your portfolio site, plus two tiny endpoints so you
can add LinkedIn posts and swap your profile photo without redeploying code.

## Run it locally

```bash
pip install -r requirements.txt
export PORTFOLIO_ADMIN_KEY="pick-a-real-secret-here"
uvicorn main:app --reload
```

Visit:
- `http://127.0.0.1:8000/` — the portfolio
- `http://127.0.0.1:8000/admin` — add posts / upload your photo (enter the
  same key you set in `PORTFOLIO_ADMIN_KEY`)
- `http://127.0.0.1:8000/docs` — interactive API docs (auto-generated by FastAPI)

## Deploying

Any host that runs a Python process works: Render, Railway, Fly.io, a
VPS, etc. Set the `PORTFOLIO_ADMIN_KEY` environment variable there too —
don't leave it as `changeme` in production, or anyone can post to your site.

Start command: `uvicorn main:app --host 0.0.0.0 --port $PORT`

## About the LinkedIn integration — the honest version

You asked for "whatever I post on LinkedIn shows up automatically." That's
not something anyone can build for free right now:

- LinkedIn's API does **not** hand out a personal member's post feed to
  third-party sites. Reading posts requires their Partner Program — a
  verified company page and a review process that takes weeks — and it's
  meant for approved business integrations, not personal portfolios.
- Profile photos aren't hotlinkable outside LinkedIn's own app either;
  there's no stable public URL to pull from.
- Paid aggregators (e.g. Juicer.io) exist and *can* pull a feed
  automatically, but they cost money and still route through LinkedIn's
  restrictions — worth it only if the manual step below genuinely bothers you.

**What this backend does instead:** it gives you a 10-second manual step
that keeps everything else automatic.

1. Post on LinkedIn as normal.
2. Open `/admin`, paste the post's title, URL, date, and a short snippet.
3. It's live on your site instantly — styled to match the rest of the page,
   not an ugly embedded iframe.

Same idea for your photo: download your real LinkedIn profile photo once,
upload it at `/admin`, done. It's genuinely your photo — it just won't
silently update if you change it on LinkedIn later.

## Project structure

```
main.py              FastAPI app: serves the site + /api/posts + /api/profile
requirements.txt
data/
  posts.json          your LinkedIn post links (editable via /admin or by hand)
  profile.json         created automatically after your first photo upload
static/
  index.html           the portfolio itself
  admin.html            the /admin page
  uploads/               your uploaded photo lands here
```
